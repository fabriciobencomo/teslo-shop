# Teslo API

1. Clone Project
2. ``` yarn install ```
3. Clone The File ```.env.template ``` and rename it to  ```.env ```
4. Change the env vars
5. Up the database
```
docker-compose up -d
```

6. Up: ``` `yarn start:dev``
